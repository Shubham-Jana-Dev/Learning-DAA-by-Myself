Introduction to algorithms:-

Q1: The definiteness property of an algorithm states that

1. ﻿﻿﻿an algorithm should take less time and memory.  
2. an algorithm must terminate in finite time.  
3. ﻿﻿﻿an algorithm must be clear and unambiguous.  
4. an algorithm must produce at least one output.

Ans. C. **an algorithm must be clear and unambiguous.**

Q1: TAn algorithm is an \_\_\_\_   concept, whereas a program  is a \_\_\_\_ implementation.

1.  concrete, abstract  
2. abstract, concrete  
3. difficult, simple  
4. simple, difficult  
   

Ans. B. **abstract, concrete**

Q3: Is it true that an algorithm depends upon the size of  
the RAM?

1. ﻿﻿﻿Yes  
2. No

Ans. B. **No**

Q4: Assume that the time to execute 1 operation by a computer is 10 ms. How much time the computer will  
take to execute 3\*\*n operations where n \= 5?

1. 243 ms  
2. 243 ms  
3. 9 seconds  
4. 2.43 seconds

Ans. D. **2.43 seconds 10\*3\*\*5 \= 2,340 ms \= 2.43 seconds**

Q5: In priori analysis, we \_\_\_\_ time and memory space.

1. estimate  
2. Calculate  
3. Measure  
4. ignore

Ans. A. **estimate**

Q6: A decidable problem is a problem for which\_\_\_\_

|  | a) no efficient algorithm |
| :---- | :---- |
|  | b) efficient algorithm |
|  | c) no algorithm |

Q7: Which of the following statements best describes  
posteriori analysis?

1. ﻿﻿﻿It is based on theoretical predictions.  
2. ﻿﻿﻿It involves analyzing the algorithm's performance before implementation.  
3. ﻿﻿﻿It relies on actual performance data collected by running the algorithm.  
4. ﻿﻿﻿It is independent of the algorithm's actual performance.

Ans. C. **It relies on actual performance data collected by running the algorithm.**

Q10: Which of the following are the characteristics of an  
algorithm?
